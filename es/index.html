Audiciones VTuber

La agencia de VTuber "Hololive Production" actualmente no está realizando audiciones para aquellos que tienen experiencia en distribución de contenido (juegos, canciones, creadores) y les gustaría desafiarse a sí mismos como VTuber proporcionando la base para un personaje original para apoyar sus actividades de VTuber. ¡Estén atentos para cuando abramos la próxima ronda de audiciones!

"Shiny Smily Story"

La primer canción original de Hololive, "Shiny Smily Story".

Esta canción, que tiene sonidos brillantes, pop y enérgicos en las letras que describen las actividades y sentimientos de cada sueño, es una canción que se puede decir que es el tema principal de Hololive con diversos talentos de afiliación...
